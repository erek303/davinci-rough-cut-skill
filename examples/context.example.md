# Project Context

Audience: people who want a concise educational video.

Main story: explain the core idea, then keep concrete examples that make the idea easier to understand.

Must keep:

- clear thesis statements
- specific examples
- emotional or surprising turns

Can remove:

- filler
- repeated setup
- off-topic discussion

Tone: clear, practical, and human.

Target length: 30-50% of the original timeline.
